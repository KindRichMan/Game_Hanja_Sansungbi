# 한자성어 별똥별

한자성어 별똥별은 밤하늘에서 떨어지는 한자성어를 보고, 제한 시간 안에 해당 한자성어의 뜻을 입력하는 학습형 타자 게임입니다. 정답을 입력하면 점수를 얻고, 한자가 감점선까지 내려오면 에너지가 줄어듭니다.

## 게임 방식

1. 닉네임을 입력합니다.
2. 시간 제한을 선택합니다.
3. `START` 버튼을 누르면 한자가 위에서 아래로 떨어집니다.
4. 아래 입력창에 한자의 뜻을 입력하고 Enter를 누릅니다.
5. 에너지가 0이 되거나 제한 시간이 끝나면 게임이 종료됩니다.
6. 게임 종료 후 점수가 저장되고 TOP 10 랭킹이 표시됩니다.

## 주요 기능

- 별똥별 낙하형 한자성어 타자 게임
- 2분 제한 모드와 무제한 모드
- 10단계 난이도 진행
- 단계별 한자성어 묶음 출제
  - 1~2단계: 1~2단계 묶음
  - 3~4단계: 3~4단계 묶음
  - 5~7단계: 5~7단계 묶음
  - 8~10단계: 전체 한자성어 무작위
- 단계별 속도와 한자 출현 간격 조절
- 2분 모드 단계별 통과 목표
  - 1단계: 10개
  - 2단계: 12개
  - 3단계: 10개
  - 4단계: 12개
  - 5단계: 10개
  - 6단계: 12개
  - 7단계: 14개
  - 8단계: 10개
  - 9단계: 15개
  - 10단계: 20개
- 단계별 속도 묶음
  - 1~2단계: 기존 1단계 속도
  - 3~4단계: 기존 2단계 속도
  - 5~7단계: 기존 3단계 속도
  - 8~10단계: 기존 4단계 속도
- 에너지 시스템
- 콤보 표시
- 노란 보너스 한자
  - 행운: 별똥별 효과와 함께 모든 한자 제거
  - 행운: 별똥별 효과와 함께 다음 단계 즉시 진입 및 100점 추가
  - 꽝: 폭탄 낙하 효과와 함께 5초 동안 한자 가리기
- 닉네임별 최고 점수 저장
- TOP 10 랭킹 표시
- 모바일 화면과 키보드 대응
- PWA 캐시 지원
- 브라우저 단어 캐시로 반복 실행 시 Firestore 읽기 절감
- 최고점과 랭킹 캐시로 불필요한 Firestore 조회 절감
- 관리자 페이지를 통한 단어 관리
- CSV 업로드를 통한 단어 일괄 등록

## 학습에 도움이 되는 점

- 떨어지는 한자를 빠르게 보고 뜻을 떠올리게 하여 회상 훈련에 도움이 됩니다.
- 제한 시간과 단계 목표가 있어 반복 학습의 집중도를 높일 수 있습니다.
- 점수와 랭킹을 통해 학습 동기를 유지할 수 있습니다.
- 관리자 페이지에서 학습 단어를 계속 추가할 수 있어 개인별 학습 범위를 확장하기 쉽습니다.
- 모바일에서도 사용할 수 있어 짧은 시간에 반복 학습하기 좋습니다.

## 데이터 구조

- `words`: 게임에 출제되는 한자 단어
  - `hanja`: 화면에 표시되는 한자
  - `reading`: 사용자가 입력해야 하는 뜻
  - `stageGroup`: 출제 단계 묶음
- `ranking`: 닉네임별 최고 점수
  - `nickname`: 닉네임
  - `score`: 최고 점수
  - `updatedAt`: 갱신 시각
- `logs`: 관리자 로그인 관련 로그

## 주요 파일

- `index.html`: 게임 화면과 게임 로직
- `admin.html`: 관리자 페이지
- `firebase.js`: Firebase 초기화와 Firestore 접근 함수
- `firestore.rules`: Firestore 보안 규칙
- `firebase.json`: Firebase Hosting 및 Firestore Rules 배포 설정
- `sw.js`: 서비스 워커 캐시 설정
