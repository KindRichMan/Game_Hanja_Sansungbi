# 한자 산성비 게임 정리

## 게임 개요

한자 산성비 게임은 화면 위에서 떨어지는 한자를 보고, 아래 입력창에 해당 한자의 뜻을 입력해 맞히는 웹 게임입니다. 단어가 감점선까지 내려오기 전에 정답을 입력해야 하며, 놓치면 에너지가 줄어듭니다.

## 주요 기능

- 한자 단어 낙하형 타자 게임
- 닉네임 입력 후 점수 기록
- 1분 제한 모드와 무제한 모드
- 단계별 속도와 출현 간격 조절
- 에너지 시스템
- 콤보 표시
- 보너스 한자 효과
- 게임오버 후 TOP 10 랭킹 표시
- 모바일 키보드와 화면 높이 대응
- PWA 서비스 워커 기반 캐시
- Firebase Firestore 기반 단어, 랭킹 저장
- 관리자 페이지를 통한 단어 조회, 검색, 추가, 수정, 삭제
- CSV 업로드를 통한 단어 일괄 등록

## 기대 효과

- 한자의 뜻을 반복적으로 입력하면서 자연스럽게 암기할 수 있습니다.
- 제한 시간과 낙하 속도 때문에 단순 암기보다 빠른 회상 훈련에 도움이 됩니다.
- 랭킹 기능으로 학습 동기를 높일 수 있습니다.
- 관리자 페이지에서 단어를 계속 추가할 수 있어 학습 범위를 확장하기 쉽습니다.
- 모바일에서도 사용할 수 있어 짧은 시간에 반복 학습하기 좋습니다.

## 데이터 구조

- `words`: 게임에 출제되는 한자 단어 목록
  - `hanja`: 화면에 표시되는 한자
  - `reading`: 사용자가 입력해야 하는 뜻
- `ranking`: 닉네임별 최고 점수
  - `nickname`: 닉네임
  - `score`: 최고 점수
  - `updatedAt`: 갱신 시각
- `logs`: 관리자 로그인 관련 로그

## 보안 개선 내용

- Firestore Security Rules를 추가해 단어 수정 권한을 관리자 이메일로 제한했습니다.
- 랭킹은 공개 읽기를 허용하되, 허용된 필드와 점수 범위를 제한했습니다.
- 관리자 목록과 랭킹 출력에서 `innerHTML` 사용을 제거해 HTML 삽입 위험을 줄였습니다.
- 단어 로딩 실패와 점수 저장 실패 시 사용자에게 오류 메시지를 보여주도록 했습니다.

## 배포 관련 파일

- `index.html`: 게임 화면과 게임 로직
- `admin.html`: 관리자 페이지
- `firebase.js`: Firebase 초기화와 Firestore 접근 함수
- `firestore.rules`: Firestore 보안 규칙
- `firebase.json`: Firebase Hosting 및 Firestore Rules 배포 설정
- `sw.js`: 서비스 워커 캐시 설정
